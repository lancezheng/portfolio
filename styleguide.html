Grid

This site uses a flexible 12-column grid system. Simply a add div.row that contains div.col with .col--{number} to establish size of the column. For example, a div.col.col--3 creates a column that fills 1/4 of the row. There are up to 12 column sizes to choose from.

Adaptive grid: Grid can also be applied based on breakpoint by using .col--{number}{breakpoint}. For example, .col--4md will ensure a 1/3 column width at md breakpoint. The following breakpoints are available:

  • md > 560px
  • lg > 720px
