Forage

  • My Role: interaction design, Interface design

  • Date: September - December 2017

  • Type of project: Leisure

Description of the project

This is a project that I created for fun. The goal of the project is to design a series of mobile app interfaces for the Forage restaurant.  The app helps customers acquire more information about the restaurant menu, make reservations and be notified of upcoming events. I aim to foster customer loyalty and increase the number of customers using the app.

Project Process

created style guide - interaction design - generated user interface

Created the style guide

According to the Forage website that they already have, I selected the fonts and styles that they currently have and sort them into the style guide.

I was struggling if I need to match my app with the website because the color style seems not my type. But When I tried to to create several pages then I can find harmony in it.

Interaction design 

Through interaction, I wanted to engage the customer. They can check the menu, book a table, write reviews, check previous and upcoming events, and manage their accounts through the app.

Generated interface

After the interaction design, I transformed all the ideas into user interface using Sketch.  

Retrospective

This fun project allowed me to further my knowledge and skills in app interface design.
